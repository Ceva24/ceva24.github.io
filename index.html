About

Gift Grabber is based on the classic Commodore Amiga game Flag Catcher.

Created by Chris Evans, 2015 using Bootstrap and Backbone.js.
View the source code on Github!

Rules

Find the gift tile!
Arrows indicate the general direction the gift tile is in
Numbers represent how far away the gift tile is (Doesn't include diagonal movement!)
Watch out for fires, they will hide all other revealed tiles on the board!
The number and location of tiles varies from game to game, so some games will be harder than others!
